Detailed Description

A robot szimulátor üzenet .

Tartalmazza az összes mezőt, ami az üzenetekben található. A továbbiakban a beérkezett JSON formátumú üzeneteket a parser egy ilyen objektumba írja Valamint a GUI és a kommunikációs réteg között az adatok cseréje "ezzel történik"
