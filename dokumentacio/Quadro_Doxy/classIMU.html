Member Function Documentation

void IMU::KalmanFiltering ( )

This function calls the register reads to get the gyroscope, and accelerometer datas, and analyzes the readed datas The method to minimalize the drifft, of the data, and get a better orientation data is called to Kalman filtering

To process the readed gyroscope data you need a known sampling time, it in this program it is set to 100 Hz, to set the time check the Timer class

As a template i used this example, for further information, please check this link: https://github.com/TKJElectronics/KalmanFilter/blob/master/examples/MPU6050/MPU6050.ino
